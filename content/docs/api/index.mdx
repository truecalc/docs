This tab documents the machine-facing surfaces of TrueCalc:

- **MCP** — the `truecalc-mcp` server tools (for Claude Desktop, Claude Code,
  and other MCP clients). Pages here will be generated from the MCP tool schemas.
- **REST** — the hosted API (planned; generated from `openapi.json` via
  fumadocs-openapi when the hosted surface ships).
- Library packages: [`truecalc-core` on crates.io](https://crates.io/crates/truecalc-core)
  and [`@truecalc/core` on npm](https://www.npmjs.com/package/@truecalc/core).
