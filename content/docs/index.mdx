TrueCalc is a spreadsheet formula engine with Google Sheets conformance, for
Rust, JavaScript/WASM, MCP, and (soon) a hosted API.

- **[Tutorials](./tutorials/index.mdx)** — hands-on, build-it-yourself
  walkthroughs. Every step runs in your browser. Start here if you're new.
- **[Learn](./learn/index.mdx)** — a beginner→advanced guide to spreadsheet
  formulas and workbooks with TrueCalc.
- **[Reference](./reference/index.mdx)** — a page for every function (with a
  runnable widget), plus error codes, dialects, and the workbook schema.
- **[Compare](./comparisons/index.mdx)** — side-by-side comparisons like
  VLOOKUP vs XLOOKUP vs INDEX/MATCH, with runnable examples.
- **[By industry](./industries/index.mdx)** — functions grouped by the work you
  do: finance, operations, science, and more.
- **[API](./api/index.mdx)** — MCP tools and the hosted REST API.

New to spreadsheet formulas? Follow the
**[Beginner path](./paths/beginner.mdx)** for an ordered journey.
