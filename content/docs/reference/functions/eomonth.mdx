## When to use it

### What does EOMONTH return?

EOMONTH returns the date of the last day of a month, offset by the number of months you specify. A zero offset gives the end of the start date's own month.

### How do I get the end of the current month or a future month?

Pass 0 as the second argument for the same month, a positive number for months ahead, or a negative number for months behind the start date.
